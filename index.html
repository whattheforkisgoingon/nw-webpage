About

Network Monitor is a lightweight tool for Windows designed to monitor your network connection by sending periodic ping requests. View statistics like average latency, packet loss rate, and more, all in a user-friendly interface.

Features include:

  • Customizable ping interval and timeout.
  • Real-time latency and packet loss tracking.
  • Support for multiple languages.
